CSS Snippet Cheatsheet

Do you ever have trouble remembering exactly what to type for complex CSS? This page is a resource for those difficult to remember code snippets! Click on any of the cards below to highlight the code, and you'll be ready to copy & paste!

Flexbox Row

Use this code to create a flexbox row.

.row {
display: flex;
flex-direction: row;
flex-wrap: wrap;
}
Flexbox Column

Use this code to create a flexbox column.

.column {
display: flex;
flex-direction: column;
}
CSS Grid Layout

Use this code to create a 12 column layout with CSS Grid.

.grid {
display: grid;
width: 100%;
grid-template-columns:
repeat (12, 1fr);
}
Linear Gradients

Use this code to create a linear gradient for your background.

.linear-gradient-background {
background: linear-gradient
(#78f4f5 0%, #097677 100%);
}
Box Transition Glow

Use this code to use transition and box shadow elements to make it glow when hovered over.

.code-card .card-header {
border-radius: 8px;
transition: all 0.5s ease-in-out;}

.code-card:hover,
.code-card:hover .card-header {
box-shadow: inset 0px 0px 8px
rgba(232, 102, 236, 1), 0 0 15px 
rgba(232, 102, 236, 1);
}
Overlay Card W/ Title

Use position properties and negative margins to raise elements higher than their natural location.

.card-header {
position: relative;
margin-top: -20px;
}
